# excalibur breakout game

The Breakout game using Excalibur.js

![demo](./imgs/demo_img.png)

## Usage

Install [Node.js](https://nodejs.org) .
Clone this repository.

```shell
npm install
```

Open 2 terminals.

In terminal A.

```shell
npm run watch
```

In terminal B.

```shell
npm run start
```

Open [127.0.0.1:5500](http://127.0.0.1:5500)

Play!!
